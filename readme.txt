
1. Ladda ner koden från Github: https://github.com/skypekitten9/ShowTracker
2. Extrahera den hämtade filen. 
3. Öppna mappen Showtracker/Showtracker i IntelliJ.
4. Se till att du kör Project SDK: 14 och Project Language level: 14
	- I IntelliJ: File - Project Structure - Project.
	- Välj rätt Project SDK och Project language level. 
	- Apply och OK. 
 
5. Lägg till de JAR-filer som finns i mappen Showtracker/Showtracker/lib
	- I IntelliJ: File - Project Structure - Libraries. 
	- Välj alla filerna i /lib och lägg till. 
	- Apply och OK. 

6. Kör klassen Controller.java som finns i mappen src/showtracker/server
	- Klicka på knappen starta server.

7. Kör klassen ClientController.java som finns i mappen src/showtracker/client
	- Skapa en användare och logga in.
